Gerador de Escala em Excel

Aplicação desktop desenvolvida em **Python** para geração automática de escalas mensais com exportação formatada em Excel.

Funcionalidades

- ✅ Cadastro de colaboradores
- ✅ Edição e exclusão de registros
- ✅ Definição de turno (Manhã, Intermediário, Tarde, Noite, Madrugada)
- ✅ Controle de folgas por dia do mês
- ✅ Período de férias (início e fim)
- ✅ Preview automático organizado por turno
- ✅ Exportação para Excel formatado
- ✅ Borda externa grossa por bloco de turno
- ✅ Freeze panes automático
- ✅ Padrão visual profissional (Cambria 11)

Tecnologias Utilizadas

- Python 3
- ttkbootstrap (interface moderna)
- openpyxl (geração de Excel)
